## [3.0.1] - 2026-04-22

### Fixed
- 공유 링크 무효 문제 해결: `share-encoder`의 base64 출력에 포함된 `+` 문자가 쿼리 파라미터에서 공백으로 디코딩되어 `?v3data=` 복원이 실패하던 버그를 URL-safe base64(`+→-`, `/→_`, trailing `=` 제거)로 수정. 회귀 테스트 2건 추가.

---

## [3.0.0] - 2026-04-19

### Added
- 4-테이블 정규화 DB 기반 데이터 엔진 재설계: `games`/`bosses`/`schedules`/`settings` 4-테이블 구조로 재구축하여 데이터 일관성과 무결성 강화
- 자정 이후 '오늘+내일' 스케줄 자동 재구축(48시간 자동 확장)
- 공식 프리셋 보스에 올바른 젠 주기 강제 적용
- 공유 URL v3data 포맷: base64 JSON 기반으로 더 안정적이고 확장 가능한 공유 URL로 전환

### Changed
- 텍스트 모드 제거, 폼 기반 간편 입력으로 입력 방식 단일화

### Fixed
- DB 계층 방어적 프로그래밍 보강: QuotaExceededError 등 저장 실패 명시적 보고
- import 시 필수 필드/FK 참조 스키마 무결성 검증
