# 🔔 보스 알리미 (Boss Alarm)
### "보스 알림을 자동으로 해주는 그런 서비스 없어?" - 보스 알리미가 그 정답입니다.

![Boss Alarm Screenshot](https://i.postimg.cc/gcZddxqV/screencapture-127-0-0-1-5500-index-html-2025-12-01-15-48-11.png)

보스 알리미는 오딘, 리니지 등 모든 MMORPG 유저를 위한 **가장 스마트한 자동 보스 알림 및 보탐 매니저**입니다. 복잡한 설치나 회원가입 없이 브라우저에서 바로 사용 가능한 강력한 게임 보조 도구입니다.

---

## ✨ 5가지 핵심 강점 (Key Highlights)

*   **🚀 자동 보스 알림 시스템**: 젠 주기만 설정하면 다음 출현 시간을 자동으로 추적하고, 배너와 음성(TTS)으로 정확한 타이밍에 알려줍니다. (5분 전/1분 전/정시 알림)
*   **🖥️ PIP 미니 타이머 (Picture-in-Picture)**: 게임 화면 위에 작은 위젯 창으로 띄워두세요. 창을 최소화해도 보스 남은 시간을 실시간으로 확인할 수 있습니다.
*   **🔗 실시간 시간표 공유**: 클릭 한 번으로 고유 짧은 URL을 생성하여 우리 팀이나 길드원에게 현재 보스 시간표를 즉시 공유할 수 있습니다.
*   **⏱️ 스마트 젠 계산기 & 광 계산기**: 복잡한 암산 없이 **남은 시간만 입력**하면 즉시 다음 젠 시간이 계산됩니다. 필드 보스 '광' 타이밍도 완벽하게 추적합니다.
*   **🛠️ 커스텀 보스 관리**: 기본 제공 보스 외에도 나만의 보스 이름과 젠 주기를 자유롭게 추가하고 서버별로 독립된 작업 공간을 유지할 수 있습니다.

---

## 📱 기기별 최적화 경험 (Device Specifics)

*   **💻 PC 환경**: 사이드바 기반의 넓은 레이아웃과 PIP 미니 위젯, 시스템 배너 알림으로 멀티태스킹에 최적화되어 있습니다.
*   **📱 모바일 환경**: 하단 탭 내비게이션 바를 통해 대시보드와 스케줄러를 한 손으로 쉽게 조작할 수 있습니다.
*   **💬 카카오톡 탈출 지원**: 인앱 브라우저의 알림 제한을 극복하기 위해 크롬/사파리 등 외부 브라우저로의 자동 이동 기능을 제공합니다.

---

## 🛠️ 주요 기능 상세 (Feature Details)

1.  **대시보드**: 다음 보스 일정을 한눈에 확인하고 PIP 모드를 실행합니다.
2.  **보스 시간표**: 등록된 보스 목록을 테이블이나 카드 형태로 깔끔하게 조회합니다.
3.  **보스 스케줄러**: 텍스트 모드 일괄 입력 또는 간편 입력 모드를 통해 수십 개의 보스 시간을 순식간에 업데이트합니다.
4.  **계산기**: 젠 계산기와 광 계산기 기능을 통해 보스 젠 타임을 1초의 오차 없이 산출합니다.
5.  **설정**: 요일별 반복되는 고정 알림(월드 보스 등)을 관리하고 사운드 볼륨을 세밀하게 조절합니다.
6.  **릴리즈 노트**: 최신 업데이트 내용을 데이터 기반의 알림 모달을 통해 빠르게 안내받습니다.
