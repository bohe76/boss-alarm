---
description: 변경 사항을 분석하여 커밋 메시지를 생성하고 커밋을 수행합니다.
---

1. `git diff --staged`를 분석하여 변경 사항을 파악합니다.
2. `GEMINI.md`의 **[Git 커밋 작업 원칙]** 및 **[커밋 메시지 유형]**을 준수하여 한글 커밋 메시지를 생성하고 사용자에게 제안합니다.
3. 사용자에게 메시지 승인을 요청합니다.
4. 승인 시 `GEMINI.md`의 **[커밋 실행 절차]**에 따라 `commit_message.txt` 임시 파일을 활용하여 커밋을 수행합니다.
5. 커밋이 완료되면 임시 파일을 삭제하고 보고한 뒤 대기합니다.
