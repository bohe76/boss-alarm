# 보스 알리미 - 화면 구성 기획 문서

## 1. 전체 레이아웃

애플리케이션은 다음과 같은 주요 영역으로 구성됩니다.
-   **헤더 (Header)**: 상단 고정 영역
-   **내비게이션 메뉴 (Sidebar)**: 좌측 접힘/펼쳐짐 가능한 메뉴
-   **메인 콘텐츠 영역 (Main Content Area)**: 선택된 메뉴에 따라 내용이 동적으로 변경되는 주 영역
-   **푸터 (Footer)**: 하단 고정 영역

## 2. 헤더 (Header)

-   **구성 요소**:
    -   **햄버거 메뉴 아이콘**: 좌측 정렬. 사이드바를 토글하는 역할.
    -   **앱 제목 "🎙️ 보스 알리미"**: 중앙 정렬.
    -   **알람 ON/OFF 버튼**: 우측 정렬. 알람 기능의 활성화/비활성화 상태를 표시하고 제어.

## 3. 내비게이션 메뉴 (Sidebar)

-   **특징**: 접힘/펼쳐짐 가능한 구조.
-   **메뉴 항목 (아이콘 + 텍스트 라벨)**:
    -   대시보드 (Dashboard)
    -   보스 설정 (Boss Settings)
    -   젠 계산기 (Zen Calculator)
    -   공유 (Share)
    -   알림 로그 (Alarm Log)
    -   알림 설정 (Notification Settings)
    -   도움말 (Help)
    -   릴리즈 노트 (Version Info)

## 4. 메인 콘텐츠 영역 (Main Content Area)

### 4.1. 화면: 대시보드 (Dashboard)

-   **제목**: "대시보드"
-   **다음 보스 표시**:
    -   라벨: "다음 보스"
    -   내용: "[보스 시간] [보스 이름] ([남은 시간])" (예: "21:43 화신그로아 (00:37:52)")
-   **다가오는 보스 목록**:
    -   제목: "다가오는 보스"
    -   내용: 다가오는 보스 10개 목록 ([보스 시간] [보스 이름] ([남은 시간]) 형식)
-   **알림 상태 요약**:
    -   라벨: "알림 상태"
    -   내용: "알림 실행 중" 또는 "알림 중지됨"
-   **최근 알림 로그**:
    -   라벨: "알림 로그"
    -   내용: 최근 알림 로그 항목 1개 표시

### 4.2. 화면: 보스 설정 (Boss Settings)

-   **제목**: "보스 설정"
-   **보스 목록 입력 텍스트 영역**: 사용자가 동적 보스 목록을 입력하고 수정.

### 4.3. 화면: 젠 계산기 (Zen Calculator)

-   **제목**: "젠 계산기"
-   **남은 시간 입력 필드**: `HH:MM` 또는 `HH:MM:SS` 형식으로 남은 시간을 입력.
    -   플레이스홀더: "예: 00:57:23 또는 00:05"
-   **보스 출현 시간 표시 영역**: 계산된 보스 출현 시간을 `HH:MM:SS` 형식으로 표시.
    -   초기값: "--:--:--"
-   **기능**: 입력 필드에 `HH:MM` 또는 `HH:MM:SS` 형식의 유효한 시간이 입력되면 실시간으로 현재 시간과 합산하여 보스 출현 시간을 계산하고 표시. `SS`가 생략된 경우 `00`으로 처리.

### 4.4. 화면: 알림 설정 (Notification Settings)

-   **제목**: "알림 설정"
-   **고정 알림 섹션**:
    -   제목: "고정 알림"
    -   **고정 알림 카드 목록**:
        -   각 카드: [보스 시간] [보스 이름], 활성화/비활성화 토글 스위치, '편집' 버튼, '삭제' 버튼.
    -   **새 고정 알림 추가 섹션**:
        -   '시간' 입력 필드 (HH:MM 형식)
        -   '이름' 입력 필드
        -   '추가' 버튼

### 4.4. 화면: 알림 로그 (Alarm Log)

-   **제목**: "알림 로그"
-   **로그 가시성 토글**: 로그 표시/숨김 제어.
-   **로그 컨테이너**: 스크롤 가능한 영역에 로그 메시지 표시.

### 4.5. 화면: 공유 (Share)

-   **제목**: "공유"
-   **설명 텍스트**: "현재 설정된 보스 목록을 다른 사람과 쉽게 공유 할 수 있는 고유 링크를 생성합니다"
-   **공유 메시지 표시**: 링크 생성 과정 및 결과 메시지 (예: "링크가 생성 중 입니다. 잠시만 기다려 주세요..", "클립 보드에 복사 되었습니다.")
-   **공유 기능 동작**: '공유' 화면으로 이동 시, 자동으로 현재 설정된 보스 목록과 고정 알림을 기반으로 단축 URL을 생성하고, 생성 완료 시 클립보드에 복사됩니다. 이 과정은 화면에 표시되는 메시지를 통해 사용자에게 전달됩니다.
