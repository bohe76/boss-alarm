기능 설명서

1. 보스 목록 관리
텍스트 영역에 보스 출현 시간과 이름을 입력하고 수정할 수 있습니다. 보스 목록은 시간 순서대로 입력해야 합니다.

각 보스 항목은 'HH:MM 보스이름' 또는 'HH:MM:SS 보스이름' 형식으로 한 줄에 입력합니다. 초 단위 입력 시 더 정밀한 알람이 가능하며, 생략 시 00초로 처리됩니다.
날짜를 지정하려면 'MM.DD' 형식으로 별도의 줄에 입력합니다. 이 날짜는 해당 날짜의 첫 번째 보스 항목 전에 위치해야 하며, 다음 'MM.DD' 줄이 나타나거나 목록이 끝날 때까지 모든 후속 보스 항목에 적용됩니다.

동적 보스 목록의 경우, 알림이 울린 보스는 목록에서 즉시 삭제됩니다. 고정 알림은 삭제되지 않고 매일 반복됩니다.

예시:
11.13
11:00 오딘
23:00:45 발리
11.14
00:05 흐니르

2. 젠 계산기
남은 시간을 입력하여 보스 출현 시간을 계산하는 기능입니다.
- "젠 계산기" 메뉴를 통해 접근할 수 있습니다.
- 남은 시간 입력 필드에 'HH:MM' 또는 'HH:MM:SS' 형식으로 시간을 입력하면 실시간으로 보스 출현 시간이 계산되어 표시됩니다.
- 유효하지 않은 형식의 입력 시 '--:--:--'로 표시됩니다.

3. 보스 스케줄러
게임별 보스 목록을 기반으로 남은 시간을 입력하여 젠 시간을 계산하고, 이를 보스 설정에 적용할 수 있는 기능입니다.
- "보스 스케줄러" 메뉴를 통해 접근할 수 있습니다.
- '게임 선택' 드롭다운에서 게임을 선택하면 해당 게임의 보스 목록이 동적으로 표시됩니다.
- 각 보스 옆의 입력 필드에 남은 시간을 입력하면 실시간으로 예상 젠 시간이 계산되어 표시됩니다.
- '남은 시간 초기화' 버튼을 클릭하면 모든 보스의 남은 시간 입력 필드가 초기화됩니다 (확인 대화 상자 포함).
- '보스 설정 적용' 버튼을 클릭하면 현재 입력된 유효한 보스 정보가 정렬되어 "보스 설정" 화면의 텍스트 영역으로 자동 입력됩니다. 이와 동시에 대시보드에도 업데이트된 보스 목록이 반영됩니다.

4. 시간 기반 알림
보스 출현 5분 전, 1분 전, 그리고 정확한 출현 시간에 알림이 울립니다.

5. 오디오 알림
웹 음성 API를 활용하여 음성 알림을 제공합니다. 알림음은 시스템 설정에 따라 재생됩니다.

6. 알림 로그
"알림 로그" 섹션에서 시스템 메시지 및 트리거된 알림을 모니터링할 수 있습니다.

7. 공유 링크 생성
보스 목록을 수정한 후 "공유 링크 생성 (Short URL)" 버튼을 클릭하면, 현재 보스 목록을 인코딩하여 공유 가능한 짧은 URL이 생성됩니다. 이 URL을 다른 사람들과 공유하여 동일한 보스 목록을 쉽게 불러올 수 있습니다.

공유 링크에는 텍스트 영역에 입력된 동적 보스 목록과 '알림 설정' 화면에서 추가/설정된 고정 알림 목록이 모두 포함됩니다.

예시:
1. 텍스트 영역에 보스 목록을 입력합니다.
2. "공유 링크 생성 (Short URL)" 버튼을 클릭합니다.
3. 생성된 짧은 URL을 복사하여 공유합니다.

8. 고정 알림
고정 알림은 매일 반복적으로 알림을 받고 싶은 보스에 대해 설정할 수 있습니다. '알림 설정' 화면에서 고정 알림을 관리할 수 있습니다.

각 고정 알림은 카드 형태로 표시되며, 보스 이름, 시간, 활성화/비활성화 토글, 편집 및 삭제 버튼을 포함합니다.

**8.1. 새 고정 알림 추가**
1. '알림 설정' 화면 하단의 "새 고정 알림 추가" 섹션으로 이동합니다.
2. '시간' 입력 필드에 HH:MM 형식으로 시간을 입력합니다 (예: 08:30).
3. '이름' 입력 필드에 보스 이름을 입력합니다 (예: 월드 보스).
4. '추가' 버튼을 클릭합니다.
5. 새로 추가된 고정 알림이 목록에 표시됩니다.

예시:
*   시간: 08:30, 이름: 월드 보스 -> '추가' 버튼 클릭

**8.2. 고정 알림 편집**
1. 편집하려는 고정 알림 카드에서 '편집' 버튼을 클릭합니다.
2. 시간 또는 이름 입력 필드가 활성화되어 수정할 수 있게 됩니다.
3. 원하는 대로 시간이나 이름을 변경합니다.
4. '저장' 버튼을 클릭하여 변경 사항을 적용합니다.

예시:
*   기존 알림: 08:30 월드 보스 -> '편집' 버튼 클릭
*   시간: 09:00, 이름: 필드 보스 -> '저장' 버튼 클릭

**8.3. 고정 알림 삭제**
1. 삭제하려는 고정 알림 카드에서 '삭제' 버튼을 클릭합니다.
2. 해당 고정 알림이 목록에서 즉시 제거됩니다.

**8.4. 고정 알림 활성화/비활성화**
1. 각 고정 알림 카드에 있는 토글 스위치를 사용하여 해당 알림을 활성화하거나 비활성화할 수 있습니다.
2. 비활성화된 알림은 알림이 울리지 않으며, 카드 배경이 흐리게 표시됩니다.
