# 이슈 011: 헤더 알람 아이콘 색조 회전 애니메이션 미작동

## 1. 문제점

헤더의 알람 아이콘(SVG path 기반)이 ON 상태일 때, CSS `filter: hue-rotate()` 애니메이션이 적용되지 않고 아이콘이 고정된 빨간색으로만 표시됩니다.

-   `filter: hue-rotate(120deg);`와 같은 정적 필터는 정상 작동하여 색상이 변하는 것을 확인했습니다. 이는 `filter` 속성 자체는 유효하다는 것을 의미합니다.
-   문제는 CSS `animation`과 `filter` 속성이 SVG `stroke`에 함께 적용될 때 발생하고 있습니다.

## 2. 제안된 해결책 (초기 검토)

이 문제를 해결하기 위해 다음과 같은 접근 방식을 고려할 수 있습니다.

-   **SVG 구조 재검토:** `filter`가 `<svg>` 요소가 아닌 `<path>` 요소에 직접 적용되어야 하는지 확인. (Google 검색 결과, 애니메이션을 `<path>`에 직접 적용하는 것이 더 안정적일 수 있음을 시사)
-   **CSS 애니메이션 기법 변경:** `filter` 애니메이션 대신 `stroke` 속성 자체를 `HSL` 값을 사용하여 직접 애니메이션하는 방법 검토.
-   **JavaScript 제어:** CSS만으로 해결이 어렵다면 JavaScript를 통해 애니메이션을 제어하는 방법도 고려.

## 3. 결론 및 다음 단계

-   SVG 필터 애니메이션이 `<svg>`와 `<path>` 요소 간에 어떻게 상호작용하는지에 대한 명확한 이해가 필요합니다.
-   `<path>` 요소에 직접 애니메이션을 적용하는 방식으로 구현을 시도하여 문제가 해결되는지 확인합니다.
-   `HSL` 값을 이용한 `stroke` 속성 직접 애니메이션도 대안으로 고려합니다.
