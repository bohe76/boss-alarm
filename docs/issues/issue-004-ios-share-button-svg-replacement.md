# 이슈-004: iOS PWA 설치 안내 모달 내 공유 버튼 SVG 교체 필요

## 문제점

iOS PWA 설치 안내 모달에 사용된 '공유' 버튼 아이콘은 현재 탭 메뉴에서 사용하는 일반 공유 아이콘 SVG를 재활용하고 있습니다. 이 아이콘은 PWA 설치 흐름에서 Safari의 네이티브 '공유' 버튼을 시각적으로 안내하는 용도로 사용되나, iOS 네이티브 '공유' 버튼 아이콘과는 시각적 형태가 다릅니다.

## 해결 방안

향후 적절한 시점에 iOS 네이티브 '공유' 버튼의 형태와 유사한 SVG 아이콘을 제작하거나 찾아, 현재 `index.html`에 인라인으로 삽입된 SVG 코드를 교체해야 합니다. 이는 사용자에게 보다 직관적인 시각적 안내를 제공하고, 앱의 사용자 경험을 개선할 것입니다.

## 관련 파일

- `index.html` (현재 인라인 SVG 삽입 위치)
- `images/` (새로운 SVG 아이콘 파일 저장 위치 고려)
