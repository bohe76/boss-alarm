# Unreleased Changes

이 파일은 다음 버전 업데이트에 포함될 변화된 기능들을 기록하는 곳입니다.
업데이트가 완료되면 이 내용은 전체 버전 기록으로 이동됩니다.

## [새로운 기능 및 개선 사항]

### 편리해진 보스 관리와 자동 스케줄러
- **메뉴 명칭 변경 및 기능 통합**: 
  - 보스 시간을 확인하는 곳은 '보스 시간표'로, 시간을 직접 입력하고 수정하는 곳은 '보스 스케줄러'로 이름을 바꾸어 용도를 더 명확히 했습니다.
  - 이제 보스 추가나 시간 수정은 모두 '보스 스케줄러' 메뉴 한곳에서 편리하게 할 수 있습니다.
- **텍스트 모드 도입 및 이동**:
  - 기존 '보스 설정'에서 사용하던 한꺼번에 입력하기 기능이 '텍스트 모드'라는 이름으로 '보스 스케줄러' 메뉴에 통합되었습니다. 
  - 리스트를 복사해서 붙여넣기만 하면 날짜별로 일정을 깔끔하게 관리할 수 있는 기존의 장점은 그대로 유지됩니다.
- **나만의 보스 젠 주기 설정**: 
  - 기본으로 제공되는 보스 외에, 사용자가 직접 보스가 다시 나타나는 시간 간격(젠 주기)을 설정할 수 있습니다. 주기만 정해주면 오늘과 내일의 전체 일정을 시스템이 알아서 계산해줍니다.
- **게임별 독립 작업 공간 (입력 정보 보존)**: 
  - 이제 '오딘' 스케줄을 짜다가 다른 게임으로 화면을 바꿔도, 작성 중이던 내용이 초기화되지 않고 그대로 유지됩니다. 여러 개의 보스 목록을 번갈아 가며 작업해도 데이터가 섞이거나 사라질 걱정이 없습니다.
- **시간대별 메모 기능 (텍스트 모드 전용)**:
  - '텍스트 모드'를 사용하면 같은 보스라도 각 시간대별로 서로 다른 메모를 남길 수 있습니다. 각 시점마다 중요한 특징이나 주의사항을 자유롭게 기록하세요.

### 똑똑해진 PC 메뉴와 모바일 화면
- **PC 메뉴 사용성 개선**: 
  - 기존의 복잡한 햄버거 버튼을 없애고, 마우스만 메뉴 위로 가져가면 메뉴 이름이 부드럽게 펼쳐지도록 개선했습니다.
  - 메뉴가 펼쳐질 때 본문 화면이 흔들리지 않도록 설계되어 더욱 쾌적한 사용이 가능합니다.
- **모바일 사용 편의성 향상**: 
  - 스마트폰에서 한 손으로도 대시보드, 시간표, 스케줄러 등 핵심 기능을 빠르게 이동할 수 있도록 화면 하단 메뉴 구성을 최적화했습니다.

### 똑똑한 자동 일정 업데이트
- **끊김 없는 '영원한 시간표' 구현**: 
  - 사용자가 링크를 통해 접속하거나 한 번이라도 시간을 입력하면, 시스템이 이를 기억했다가 매일 자정마다 내일과 모레의 시간표를 스스로 미리 생성해냅니다. 한 번의 설정만으로 나만의 시간표를 평생 중단 없이 사용할 수 있습니다.
- **업데이트 안내 및 작업 보호**: 
  - 사용자가 직접 시간을 수정하고 있을 때 일정이 바뀌면 안내창을 띄워 드립니다. 수정하던 내용을 계속 쓸지, 아니면 새로운 일정을 불러올지 직접 선택할 수 있어 작업 내용이 갑자기 사라질 걱정 없습니다.

### 안심하고 사용하는 시스템 보호
- **시스템 '자가 진단 및 복구' 엔진**: 
  - 잘못된 링크를 클릭하거나 저장된 데이터가 꼬여 앱이 멈추는 돌발 상황을 방지합니다. 문제가 감지되면 시스템이 스스로 진단하여 가장 안전한 상태로 자동 복구하므로, 어떤 상황에서도 앱을 항상 정상적으로 이용할 수 있습니다.

### 새로워진 시간표 저장 및 알림 기능
- **시간표 이미지 저장 및 텍스트 복사 기능 (신규)**: 
  - 현재 보고 있는 보스 시간표를 이미지 파일로 저장하거나 텍스트로 복사하여 다른 사람에게 공유하는 기능이 새롭게 추가되었습니다. 
- **미니 알림창(PiP) 기능 강화**: 
  - 게임 중에 띄워두는 작은 알림창 하단에 곧 나타날 보스 목록을 실시간으로 표시해 줍니다.
  - 보스 숫자가 바뀌어 알림창의 크기 조절이 필요할 때, 아이콘이 깜빡이며 알려줍니다. 알림창을 클릭하면 현재 보스 목록에 맞춰 크기가 알맞게 조절됩니다.
