# 기능 체크리스트: 보스 타이머 계산기

## 1. UI (사용자 인터페이스)
### 1.1. 내비게이션
- [ ] 사이드바에 "젠 계산기" 메뉴 항목 추가 (세 번째 위치, "보스 설정" 다음).
- [ ] "젠 계산기" 메뉴 항목에 계산기 SVG 아이콘 추가.

### 1.2. 계산기 화면
- [ ] "젠 계산기" 메뉴 선택 시 전용 화면 렌더링.
- [ ] 화면 내 "남은 시간" 입력 필드 구현.
- [ ] 화면 내 "보스 출현 시간" 표시 영역 구현.

## 2. 기능
### 2.1. 입력 처리
- [ ] "남은 시간" 입력 필드가 `HH:MM` 또는 `HH:MM:SS` 형식의 시간 지속 시간 허용.
- [ ] `SS` 부분이 생략된 `HH:MM` 형식 입력 시, `00`초로 묵시적 처리.
- [ ] `input` 이벤트 리스너를 사용하여 실시간으로 계산 트리거.

### 2.2. 계산 로직
- [ ] 입력 필드 값이 유효한 `HH:MM` 또는 `HH:MM:SS` 형식과 일치할 때만 계산 수행.
- [ ] 입력 문자열을 시, 분, 초로 정확히 파싱.
- [ ] 파싱된 지속 시간을 현재 시스템 시간에 정확히 추가.
- [ ] 결과로 생성된 미래 시간을 "보스 출현 시간"으로 설정.

### 2.3. 출력
- [ ] "보스 출현 시간"을 명확하고 사용자 친화적인 `HH:MM:SS` 형식으로 표시.
- [ ] 입력이 유효한 형식과 일치하는 경우, 사용자가 입력할 때 디스플레이 동적 업데이트.

## 3. 기술 구현
### 3.1. 새 모듈: `src/calculator.js`
- [ ] `src/calculator.js` 파일 생성.
- [ ] `calculateBossAppearanceTime(remainingTimeString)` 함수 구현 (남은 시간 문자열을 받아 계산된 출현 시간 반환).

### 3.2. 통합
- [ ] `index.html` 업데이트: 내비게이션 및 새 화면 컨테이너 추가.
- [ ] `src/dom-elements.js` 업데이트: 새 메뉴 항목, 입력 필드, 표시 영역에 대한 DOM 참조 추가.
- [ ] `src/ui-renderer.js` 업데이트: `renderCalculatorScreen()` 구현 및 `renderScreen()` 수정.
- [ ] `src/event-handlers.js` 업데이트: 내비게이션 및 입력 필드에 대한 이벤트 리스너 추가, `src/calculator.js` 호출.
- [ ] `src/style.css` 업데이트: 새 UI 요소에 대한 스타일 추가.

## 4. 오류 처리
