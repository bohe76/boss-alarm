# UI/GUI 개선 및 카드 시스템 통일 (2차) 문서

본 문서는 프로젝트의 전반적인 UI/GUI 개선 및 카드 시스템 통일 작업을 통해 이루어진 변경 사항을 기록합니다.

*   **보스 이름 색상 변경 (`#00B0FF`):**
    *   '다음 보스' 및 '다가오는 보스'(5분 미만) 섹션에서 보스 이름의 색상을 청록색 계열(`#43AA8B`)에서 더 밝은 파란색(`#00B0FF`)으로 변경하여 시인성을 높였습니다. (`src/style.css`)

*   **대시보드 로그 리스트 가독성 개선 및 용어 변경:**
    *   '알림 로그' 용어를 '로그'로 변경하여 의미를 확장했습니다. (`index.html`)
    *   대시보드 로그 리스트에서 중요 로그의 빨간색 표시를 제거하여 혼란을 줄였습니다. (`src/style.css`)
    *   가장 최근 로그 항목의 가독성을 높이기 위해 텍스트 전체를 굵게 표시하고, 청록색(`#43AA8B`)으로 색상을 변경했습니다. (`src/style.css`)
