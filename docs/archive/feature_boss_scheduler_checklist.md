## 2. 기능
### 2.1. JSON 보스 목록 로드 및 관리
- [x] `data/boss_lists.json` 파일 로드 및 파싱 로직 구현.
- [x] 게임 목록 및 해당 보스 목록을 제공하는 함수 구현.

### 2.2. 게임 선택 및 보스 목록 동적 렌더링
- [x] 게임 선택 드롭다운 변경 시, 해당 게임의 보스 목록을 동적 보스 입력 영역에 렌더링.

### 2.3. 남은 시간 입력 및 젠 시간 계산
- [x] "남은 시간" 입력 필드에 `input` 이벤트 리스너 연결.
- [x] `HH:MM` 또는 `HH:MM:SS` 형식 및 24시간 이내의 유효성 검사.
- [x] `src/calculator.js`의 `calculateBossAppearanceTime` 함수를 활용하여 젠 시간 계산.
- [x] 유효한 입력 시 계산된 젠 시간을 해당 보스의 표시 영역에 업데이트.
- [x] 유효하지 않거나 비어 있는 입력 시 표시 영역을 초기 상태(`--:--:--`)로 유지.

### 2.4. "남은 시간 초기화" 버튼
- [x] 클릭 시, 모든 "남은 시간" 입력 필드 초기화.
- [x] 클릭 시, 모든 "젠 시간" 표시 영역 초기 상태로 되돌리기.

### 2.5. "보스 설정 적용" 버튼
- [x] 클릭 시, "남은 시간" 입력 필드가 비어 있지 않고 유효한 시간을 포함하는 보스만 필터링.
- [x] 필터링된 각 보스에 대해 현재 시간과 입력된 남은 시간을 합산하여 절대적인 젠 시간(날짜 및 시간) 계산.
- [x] 계산된 젠 시간을 기준으로 보스 목록을 시간 순서대로 정렬.
- [x] 날짜가 변경될 때마다 새로운 `MM.DD` 형식의 날짜 마커 라인 삽입 로직 구현.
- [x] 정렬되고 날짜 마커가 포함된 보스 목록을 "보스 설정" 화면의 `bossListInput` 텍스트 영역에 `MM.DD` (날짜 마커) 및 `HH:MM:SS 보스이름` 형식으로 삽입.
