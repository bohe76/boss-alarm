# 기능 설명서

## 1. 보스 목록 관리

텍스트 영역에 보스 출현 시간과 이름을 입력하고 수정할 수 있습니다. 각 줄은 `HH:MM 보스이름` 형식이어야 합니다.

**예시:**
```
14:00 오딘
15:30 티르
```

## 2. 시간 기반 알림

보스 출현 5분 전, 1분 전, 그리고 정확한 출현 시간에 알림이 트리거됩니다.

## 3. 오디오 알림

웹 음성 API를 활용하여 음성 알림을 제공합니다.

## 4. 알림 로그

"알림 로그" 섹션에서 시스템 메시지 및 트리거된 알림을 모니터링할 수 있습니다.

## 5. 공유 링크 생성

보스 목록을 수정한 후 "공유 링크 생성 (Short URL)" 버튼을 클릭하면, 현재 보스 목록을 인코딩하여 공유 가능한 짧은 URL이 생성됩니다. 이 URL을 다른 사람들과 공유하여 동일한 보스 목록을 쉽게 불러올 수 있습니다.

**예시:**
1. 텍스트 영역에 보스 목록을 입력합니다.
2. "공유 링크 생성 (Short URL)" 버튼을 클릭합니다.
3. 생성된 짧은 URL을 복사하여 공유합니다.

## 6. 고정 알림

특정 보스에 대해 고정된 알림을 설정할 수 있습니다. "고정 알림" 섹션에서 토글 버튼을 사용하여 활성화/비활성화할 수 있습니다.
