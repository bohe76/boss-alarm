## 1. 개요

'보스 시간표' 화면은 현재 등록된 모든 보스의 출현 일정을 다양한 뷰(일별 목록, 표, 카드)로 표시하는 화면입니다. 동적 보스 목록과 사용자가 설정한 고정 알림을 통합하여 시간순으로 정렬하여 표시합니다. 내보내기(텍스트/이미지) 기능도 제공합니다.

보스 스케줄러에서 업데이트 완료 후 자동으로 이 화면으로 이동됩니다.

---

## 2. 사용자 시나리오

- **시나리오 1**: 사용자가 '보스 시간표' 메뉴를 클릭하면 → 현재 보스 스케줄이 일별 목록 형태로 표시된다.
- **시나리오 2**: 사용자가 뷰 전환 버튼을 클릭하면 → 일별 목록 / 표 / 카드 뷰로 전환된다.
- **시나리오 3**: "다음 보스 필터" 토글을 활성화하면 → 1초마다 갱신되어 지나간 보스가 자동으로 제거된다.
- **시나리오 4**: 내보내기 버튼을 클릭하면 → 텍스트 또는 이미지 형식으로 시간표를 내보낼 수 있다.
- **시나리오 5**: 고정 알림이 활성화된 상태에서 시간표를 보면 → 고정 알림 항목이 동적 보스와 함께 병합 표시된다.

## 4. 수용 기준 (Acceptance Criteria)

- **AC-001**: Given 보스 스케줄이 등록된 상태 / When 시간표 화면 진입 / Then 일별 목록 뷰로 보스 목록이 표시된다.
- **AC-002**: Given 고정 알림이 1개 이상 활성화된 상태 / When 시간표 화면 렌더링 / Then 고정 알림 항목이 동적 보스와 시간순으로 병합되어 표시된다.
- **AC-003**: Given "다음 보스 필터" 토글 활성화 / When 1초 경과 후 다음 보스가 변경됨 / Then 화면이 자동 갱신된다.
- **AC-004**: Given 내보내기 모달이 열린 상태 / When 1초 경과 / Then 모달이 열려있는 동안 화면 갱신이 스킵된다.
- **AC-005**: Given 텍스트 내보내기 버튼 클릭 / When 내보내기 실행 / Then 고정 알림을 포함한 전체 시간표가 텍스트로 생성된다.

## 5. 의존성

- **데이터**: `v3_schedules`(동적 보스), `v3_settings`(고정 알림)
- **모듈**: `BossDataManager.getBossSchedule()`, `LocalStorageManager.get('timetableNextBossFilter')`, `ui-renderer.js`의 `updateTimetableUI`, `renderTimetableList`, `renderExportCapture`
- **헬퍼**: `_expandFixedAlarmsInRange(rangeStart, rangeEnd)` — 고정 알림을 보스 shape으로 변환 (4경로 공유)
- **화면 진입점**: 보스 스케줄러 업데이트 완료 후 자동 이동

## 6. 비기능 요구사항

- **성능**: 내보내기 모달이 열린 동안 갱신 중지로 이미지 캡처 시 불필요한 리렌더링을 방지한다.
- **일관성**: 일별/표/카드/대시보드 4개 경로 모두 동일한 고정 알림 병합 로직(`_expandFixedAlarmsInRange`)을 사용한다.
- **스타일 확장성**: `list-item--fixed`, `boss-table-row--fixed` CSS 클래스는 향후 스타일 커스터마이징을 위해 유지한다.

## 8. 동작 명세

### 8.1. 기능 접근
사용자는 좌측 사이드바 메뉴 또는 모바일 환경의 '더보기' 메뉴에서 '보스 시간표' 메뉴 항목을 클릭하여 이동합니다. 또는 보스 스케줄러에서 업데이트 완료 후 자동으로 이동됩니다.

### 8.2. 뷰 구성
- **일별 목록 뷰**: 출현 시간과 보스 이름을 날짜별로 그룹화하여 목록 형태로 표시합니다.
- **표(Table) 뷰**: 보스 이름과 출현 시간을 표 형태로 표시합니다.
- **카드 뷰**: 보스 정보를 카드 형태로 표시합니다.

### 8.3. 고정 알림 병합
- `_expandFixedAlarmsInRange(rangeStart, rangeEnd)` 헬퍼가 활성화된 고정 알림을 `{type:'boss', isFixed:true, ...}` shape으로 변환합니다.
- 변환된 고정 알림은 동적 보스 목록과 시간순으로 병합되어 표시됩니다.
- 이 병합 로직은 일별/표/카드 뷰 및 텍스트·이미지 내보내기 4개 경로 모두에 동일하게 적용됩니다.

### 8.4. 다음 보스 필터 (자동 갱신)
- `timetableNextBossFilter` 설정이 활성화된 경우, 1초마다 현재 시각 기준 첫 번째 보스 ID를 확인합니다.
- 첫 번째 보스 ID가 변경되었을 때만 화면을 갱신합니다(불필요한 리렌더링 방지).
- 내보내기 모달이 열린 동안은 갱신을 스킵합니다.
- 화면이 비활성화되면 인터벌을 자동 해제합니다.

### 8.5. 내보내기 기능
- **텍스트 내보내기**: 현재 시간표를 텍스트 형식으로 생성합니다. 고정 알림 항목도 포함됩니다.
- **이미지 내보내기**: 현재 시간표를 이미지(캡처)로 생성합니다. 내보내기 모달에서 미리보기를 확인할 수 있습니다.
- 내보내기 시 고정 알림 48시간 병합 로직이 동일하게 적용됩니다.
