# 버전 관리 기록

## v1.1.0 (2025-11-13)
*   **대규모 리팩토링:**
    *   **모듈화 및 관심사 분리:** 모든 로직(HTML, CSS, JS)이 단일 `index.html` 파일에 응집되어 있던 구조를 기능별 모듈(예: `dom-elements.js`, `logger.js`, `speech.js`, `data-managers.js`, `boss-parser.js`, `alarm-scheduler.js`, `ui-renderer.js`, `api-service.js`, `event-handlers.js`, `default-boss-list.js`, `style.css`)로 분리하여 코드의 가독성, 유지보수성 및 확장성을 크게 향상시켰습니다.
    *   **전역 변수 캡슐화:** 전역 스코프에 선언된 DOM 요소 참조와 상태 변수들을 특정 모듈 또는 클래스 내로 이동시켜 전역 오염을 줄였습니다.
    *   **의존성 주입 개선:** 모듈 간의 직접적인 의존성을 줄이고 유연성을 높였습니다.
    *   **오류 처리 강화:** `parseBossList` 함수에 `try-catch` 블록을 추가하고, 시간 형식 및 날짜 값에 대한 유효성 검사를 강화하여 잠재적 오류에 대한 예외 처리를 개선했습니다.
    *   **음성 알림 중첩 문제 해결:** `speech.js`의 `speak` 함수에서 음성 발화 큐잉 메커니즘을 구현하여 여러 알림이 순차적으로 재생되도록 수정했습니다.

## v1.0.0 (2025-11-13)
*   **초기 릴리스:** 오딘 보스 알리미 프로젝트의 첫 번째 릴리스입니다.
    *   보스 목록 관리, 시간 기반 알림, 오디오 알림, 로깅, 공유 가능한 URL 생성 기능이 포함되어 있습니다.
    *   모든 애플리케이션 로직은 `index.html` 파일 내에 포함되어 있었습니다.
